Future Source Retrieval

* A future source retrieval request on a without-source project will use the following calculation for the source retrieval quote:

Total the without-source prices of all project cycles that were made to reach the final product. Divide this number by 0.6 to find a larger estimated total. Calculate the difference between these values to obtain the payable difference.

Source retrieval uses a different pricing because when we initiate without-source projects, we are able to develop more efficiently with code libraries. At source retrieval request, there are overheads involved in extracting your necessary compilable code from our infrastructure, as well as considerations on the value of source code and changed internal client classification type from trader to product developer. Quotation method is subject to change with the economy.

More on How We Quote

When we quote, we place an internal metric on static and dynamic value of a component and generally have set pricing in increments of $8-$15.

We do this for your benefit to value outcomes with project-pricing, as opposed to encouraging slow development using hourly pricing.

However, our project-priced quoting calculations are still subject to internal metrics of consulting overhead and risk. Always request a quote and always Keep It Simple.
